Bem2d.jl - Github repository

A Julia based code for two-dimensional plane strain linear elastic boundary element (BEM) calculations (both quasi-static and quasi-dynamic) related to earthquakes and tectonics.

Blocks - Github repository

A Matlab based block model code integrating plate motions, internal deformation, and a first order approximation of earthquake cycle effects associated with the partial coupling of plate bounding faults.  Jack Loveless is the lead developer and responsible for everything good about the code!

Triangular dislocation elements - Github repository

A Matlab implementation of the closed form solutions for the volumetric strains, stresses, and displacements due to slip on a triangular dislocation element in a homogeneous elastic halfspace.  Richard Styron has developed a python version and Nikkhoo and Walter (2015) have developed a theory and implementation free of many of the singularities present in the original.

Machine learning aftershock locations - Github repository

A Python pipeline for learning (using Keras) the relationships between mainshock (from SRCMOD catalog) static elastic stress changes and aftershock locations.  Pheobe DeVries is the lead developer and any errors are due to me!
